# Rock-paper-scissors-QT

## Сборка проекта

1. Собрать сервер
    1. В папке server выполнить qmake(qmake6)
    2. После выполнить make
2. Собрать клиента
    1. В папке player выполнить qmake(qmake6)
    2. После выполнить make

Убедиться, что порт 1234 в локальной сети не занят, если он занят, то при запуске сервера указать в аргументах свободный порт. Запустить сервер. Запустить клиента (Если меняли порт у сервера, то нужно также в аргументах добавить этот порт клиентам). Играть =) 

Основные решения:
1. Создать сервер, который будет держать два сокета первых подключившихся клиента, обработывать получаемую информацию от них, определять победителя и сообщать это игрокам.
2. Создать простенькое клиентское приложение.

Что можно улучшить:
1. Можно было создать основной сервер, который бы запускал текущий сервер при запросе клиента и перенаправлял его туда, а другим клиентам сообщал бы о созданном лобби для коннекта, так можно было бы масштабировать данную игру.
2. Визуальный интерфейс клиента довольно скудный, но с предидущим улучшением, можно было бы создать целое лобби, добавить счетчики, никнеймы и т.д.
